Games

Flight over the Mountains

The yellow spheres containig the coins increase the score, the blue spheres containing the barrels fill up the fuel by 50% or 100%. The are also red spheres with the rockets inside. Going through these sphere will make the rockets available and a "rocket" button appears. You can click on that button or press the "R" key on the keyboard for using the rockets which will boost your speed. The red spheres containing the text "Game Over", should be avoided. Press the space bar or the left mouse button for tilting the elevators up, release the space bar or the left mouse button for tilting the elevators down. You can also set the throttle by pressing the "+" or "-" keys on the keyboard or by clicking on the gray bar on the right. Reduce the throttle for consuming less fuel. The game is over when the airplane touches the red sphere containing the text "Game Over" or when the airplane touches the ground.

Island

Use the airplane for going through the spheres. Reduce the throttle for consuming less fuel. The yellow spheres increase your score and the blue spheres fill up the fuel. The game is over when the airplane has no fuel anymore.

Race

This is a multiplayer game which works by using peer-to-peer connections. Up to five users can play together. Each user has to set a username. One of the users needs to create a group by clicking on "create group" and his computer will act as a server. If the group owner closes the game in the browser the game will also stop for the other users. A group number will be generated and the users have to write that group number in the input box and click on "set group". Then the users have to click on "ok". The user who has created the group has to wait until all other users are displayed as "connected" before clicking on the "ok" button. You will also need to use the airplane for going to another island. The runway of the other island is straight ahead of the runway of the current island. At the end of that runway you will find some other cars. The first player who passes through all checkpoints in the correct order wins.

Pilot

Press the arrow left or arrow right key to turn, press the arrow up key to walk. Use the space bar for jumping.

Vehicles

These sections explains how to control the vehicles in the games "Island" and "Race". The vehicle can be entered by pressing the number show on the left when the pilot is close enough. For exiting the vehicle press the "E" key.

Airplane

Use the "+" or "-" keys for setting the throttle. You can use the arrow right or arrow left keys for applying the rudder and for setting the steering wheel. Press the "S" key to set the position of the airplane to the initial position. Use the mouse for controlling the elevators and the ailerons.

Car

Press the arrow up key to accelerate and the arrow down key to brake. Use the arrow left and arrow right keys for steering. Press the "S" key to set the position of the car to last position where all wheels were on the ground. With the "L" key you can turn on and off the lights. Keep the "R" key pressed for using the reverse gear.
